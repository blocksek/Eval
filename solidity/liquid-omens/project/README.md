# Liquid Omens

The omen monks run a venue bound by strict rites, signatures, and canon. One borrower is already underwater. One reserve is still full. Their journals obsess over profitable strategies, inventory netting, and final settlement; the abbey adopted the language so enthusiastically that even routine procedures sound suspect. Something in the liturgy is less rigid than it appears.

Take the reserve.

## Objective

Drain enough `oUSD` to satisfy `Challenge.isSolved()`.

## Start

The challenge lives in `src/`.

forge build
