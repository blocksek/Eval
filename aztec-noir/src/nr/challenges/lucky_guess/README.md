# Lucky Guess

## Author
Wonderland

## Description
The Cheshire Cat challenges you to Wonderland Roshambo — five rounds of rock-paper-scissors against fate itself. The Cat draws its moves from the looking glass, an oracle of pure randomness. Or is it?

In Aztec's private execution model, oracles aren't quite what they seem. Figure out how the Cat cheats and beat it at its own game.
